### 1. Introduction to Bootstrap

#### What is Bootstrap?
- CSS Framework
- Component library
- Responsive grid system
- JavaScript plugins
- Utility classes

#### Setup Methods
1. CDN Links

```html
<link href="bootstrap.min.css" rel="stylesheet">
<script src="bootstrap.bundle.min.js"></script>
```

2. Package Managers

```bash
npm install bootstrap
```

### 2. Bootstrap Grid System

#### Grid Basics
- 12-column system
- Responsive breakpoints
- Container types
- Row and column structure

#### Grid Classes
- `.container` / `.container-fluid`
- `.row`
- `.col-*`
- `.col-{breakpoint}-*`

#### Breakpoints
- xs: < 576px
- sm: ≥ 576px
- md: ≥ 768px
- lg: ≥ 992px
- xl: ≥ 1200px
- xxl: ≥ 1400px
